# FNaF Fanpage System Architecture

This repository contains the foundational architecture contracts for the FNaF Fanpage project:

- **contracts/backend_api_contract.json**: Defines backend API endpoints and request/response schemas.
- **contracts/frontend_route_contract.json**: Specifies frontend route configuration, endpoint dependencies, and access control.
- **contracts/directory_structure.json**: Declares file mappings, ownership boundaries, entry points, and database files.

See `docs/api/README.md` for detailed endpoint usage.
