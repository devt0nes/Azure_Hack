# Cancer Awareness Website Backend

## Overview
This is the backend server for a simple cancer awareness website. It provides RESTful APIs for retrieving information about different types of cancer and handling contact form submissions.

## Features
- Fetch a list of cancer types and their descriptions.
- Submit a contact form inquiry.

## Prerequisites
- Node.js (v16 or later)
- npm (v8 or later)
- PostgreSQL (future integration)

## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd project-directory
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Create a `.env` file in the root directory and specify the following environment variables:
   ```
   PORT=3000
   ```
