# Cookies Delight

## Project Description
Cookies Delight is a website dedicated to sharing information, recipes, and fun facts about cookies. It features a recipe database, rating and commenting system, and user authentication.

## Prerequisites
- Node.js >= 18.x
- npm >= 8.x

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/your-repo/cookies-delight.git
   cd cookies-delight
   ```
2. Install dependencies:
   ```
   npm install
   ```
