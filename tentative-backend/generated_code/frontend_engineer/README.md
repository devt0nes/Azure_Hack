# Cupcake Bakery Demo Website

## Description
This is a simple, elegant demo website for showcasing cupcake offerings, store information, and providing a contact form for inquiries. The website is built using React.

## Prerequisites
- Node.js (v16 or higher)
- npm (v8 or higher)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/example/cupcake-bakery-demo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd cupcake-bakery-demo
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application
1. Start the development server:
   ```bash
   npm start
   ```
2. Open your browser and navigate to `http://localhost:3000`.

## Project Structure
- `index.html`: Main HTML file for the project.
- `styles.css`: Styling for the website.
- `main.jsx`: Entry point for the React application.
- `App.jsx`: Main application component with routing setup.
- `pages/`: Directory containing individual page components (Homepage, MenuPage, ContactPage).

## Dependencies
- React
- ReactDOM
- React Router DOM
